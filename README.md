## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone
   ```

2. Перейдите в каталог проекта:

   ```bash
   cd mindboxTestTask
   ```

3. Установите зависимости:

   ```bash
   npm install
   ```

4. Запустите приложение:

   ```bash
   npm run dev
   ```
